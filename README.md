# C3 数学库（单文件 · 无依赖 · 轻量高效）

## 简介

这是一个为 **C3 编程语言** 开发的**单文件、无任何依赖、开箱即用**的数学工具库。
包含基础运算、数组处理、数论、序列、数值限制等常用功能，适合学习、练习、小型项目直接使用。

---

## 亮点特性

- 纯 C3 编写，**单文件 math\.c3**

- **零依赖**，直接导入即可使用

- 函数齐全，覆盖新手最常用的数学 / 数组工具

- 代码简洁易读，带完整中英文注释

- 支持数组求和、平均值、排序、去重、查找、质数判断、斐波那契、范围限制等

---

## 函数功能一览表

|函数名|功能说明|
|---|---|
|`gcd\(a,b\)`|求两个数的最大公约数|
|`lcm\(a,b\)`|求两个数的最小公倍数|
|`mai\(a,b\)`|比较两个数大小|
|`maximum\(arr,\&amp;val\)`|查找数组最大值并返回位置|
|`minimum\(arr,\&amp;val\)`|查找数组最小值并返回位置|
|`oe\_num\(num\)`|判断一个数是否为偶数|
|`abs\(num\)`|求浮点数绝对值|
|`swap\(\&amp;a,\&amp;b\)`|交换两个整数的值|
|`sum\(arr\)`|数组所有元素求和|
|`product\(arr\)`|数组所有元素求乘积|
|`isPrime\(num\)`|判断一个数是否为质数|
|`sort\(arr\)`|数组快速排序（从小到大）|
|`reverse\(arr\)`|数组反转|
|`avg\(arr\)`|计算数组平均值|
|`fcl\(num\)`|计算阶乘 n\!|
|`fib\(n\)`|计算斐波那契数列第 n 项|
|`clamp\(v,min,max\)`|限制数值在指定范围内|

---

## 快速使用

1. 将 `math\.c3` 放入你的项目

2. 在主文件中导入：

```c3
module main;
import math;
```

3. 直接调用函数即可

---

## 常用示例（部分）

### 数组求和

```c3
int[] arr = [1,2,3,4,5];
int total = math::sum(arr);
```

### 数组平均值

```c3
float average = math::avg(arr);
```

### 数组去重

```c3
int[] arr = [1,2,2,3,3,3];
int[] newArr = math::unique(arr);
```

### 限制数值范围

```c3
int val = math::clamp(120, 0, 100);
```
